# RFID-Lab-System-Allocation

## Overview

Developed an RFID-based smart lab management system that automates student authentication, system allocation, and occupancy tracking using embedded hardware and software integration.

## Features

* RFID-based student identification
* Automatic FREE/OCCUPIED system allocation
* Real-time lab occupancy monitoring
* Login/logout time tracking
* Database-based student and system management

## Software Implementation

Implemented allocation logic, database handling, real-time status monitoring, and serial communication between the microcontroller and software application for automated system assignment.

## Hardware Used

* RFID Reader Module
* RFID Tags/Cards
* Microcontroller
* Embedded System Interface

## Technologies

Embedded Systems, RFID Technology, Database Management, Serial Communication, Embedded C/Arduino Programming
